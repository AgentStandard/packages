# knowledge-gap-detection

## Type
Behavioral skill — no external API or installation required. This capability is built into the agent's core system prompt and knowledge.

## What It Does
Surfaces what the agent does not know but probably should — identifies the most valuable missing context

## How to Use
This skill activates automatically based on your conversation. You do not need to install anything or configure any API keys to use it.

## When It Activates
The agent applies this skill whenever the conversation context calls for it. You can also explicitly invoke it by describing what you need.

## No Configuration Required
Behavioral skills ship with the agent's SOUL.md. They are active from the first conversation.
