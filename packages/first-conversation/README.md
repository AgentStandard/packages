## Setup Steps

### 1. Download OpenClaw (~3 min)
Go to [openclaw.ai/download](https://openclaw.ai/download) and install the app for your system (Windows, Mac, or Linux).

### 2. Get your Claude API key (~5 min)
Go to [console.anthropic.com](https://console.anthropic.com/api-keys), create a free account, and generate an API key. Copy it somewhere safe.

### 3. Add your API key to OpenClaw (~2 min)
Open OpenClaw → Settings → paste your API key.

### 4. Install your starter skills (~3 min)
Open your terminal and run:
```
openclaw skills install weather web-search
```

### 5. Say hello (~1 min)
Open a new chat and type:
> "What's the weather like in London today?"

If it answers — you're done. Welcome.
