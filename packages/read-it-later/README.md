# Read It Later

> Save it. Read it. Let it stick.

**Vertical:** Learning  
**Telegram:** Yes  
**Status:** Pending certification

## About

Save links and articles into your agent. It summarises them, extracts key ideas, and resurfaces them when relevant.

## Try it

**On Telegram (zero setup):**  
[Start Read It Later on Telegram](https://t.me/AgentStandardAI_bot?start=read-it-later)

**On OpenClaw:**
1. Copy the `telegram_system_prompt` from `agentstandard.json`
2. Paste into your agent's `SOUL.md`
3. Restart your agent
