# Wine Log

> Remember every bottle. Build a palate that's genuinely yours.

**Vertical:** Lifestyle  
**Telegram:** Yes  
**Status:** Pending certification

## About

Log wines you've tried, rate them, track what you like and why. Over time your agent builds a real picture of your palate and makes recommendations that fit how you drink.

## Try it

**On Telegram (zero setup):**  
[Start Wine Log on Telegram](https://t.me/AgentStandardAI_bot?start=wine-log)

**On OpenClaw:**
1. Copy the `telegram_system_prompt` from `agentstandard.json`
2. Paste into your agent's `SOUL.md`
3. Restart your agent
