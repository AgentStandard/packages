# Memory Architect

> An agent that remembers who you are.

**Vertical:** Productivity  
**Telegram:** Yes  
**Status:** Pending certification

## About

Structures your agent's long-term memory — daily notes distilled into lasting context, key decisions captured, important context surfaced when relevant.

## Try it

**On Telegram (zero setup):**  
[Start Memory Architect on Telegram](https://t.me/AgentStandardAI_bot?start=memory-architect)

**On OpenClaw:**
1. Copy the `telegram_system_prompt` from `agentstandard.json`
2. Paste into your agent's `SOUL.md`
3. Restart your agent
