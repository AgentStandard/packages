# VERIFY.md — Ecommerce OS

## Test Prompts

**Test 1**
> "Brief: I sell premium ceramic cookware on Shopify DTC. COGS ~£22, selling £79. Competitors: Caraway, Great Jones. Customer: women 30-45 who care about aesthetics. Run /compete ceramic cookware DTC."

**Test 2**
> "/pdp Hand-thrown ceramic mug, 350ml, dishwasher safe, matte glaze in 6 colorways. Target: design-conscious home cooks 25-40, gifting and self-purchase."

**Test 3**
> "/funnel 15,000 monthly sessions, 3.8% add-to-cart, 55% checkout initiation, 1.9% CVR. Average order £84. What's leaking and what do I fix first?"

---

## Expected Behaviour

**Test 1:** Returns competitive analysis with specific named players, their pricing, positioning angles, and review themes. Identifies gaps. Uses web search for live data.

**Test 2:** Produces a complete PDP package: SEO title, meta description, headline, 4+ benefit bullets (benefit-led not feature-led), full description paragraph, FAQ section.

**Test 3:** Benchmarks your funnel metrics against category norms. Identifies the biggest leak point. Recommends one specific fix to try first, with reasoning.

---

## You Know It's Working When...

- [ ] /compete uses web search and returns actual competitor names and price ranges
- [ ] /pdp produces all required sections (title, meta, headline, bullets, description, FAQ)
- [ ] Benefit bullets do not start with features — they start with outcomes
- [ ] /funnel gives a specific diagnosis, not "improve your conversion rate"
- [ ] Agent builds on store context from earlier in the conversation
