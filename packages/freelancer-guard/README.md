# Freelancer Guard

Read client contracts, scopes of work, and proposals before you sign them — built for freelancers, contractors, and consultants who need plain-English risk analysis without a lawyer on speed dial.

## What your agent does

- Reads contracts and flags specific risky clauses — not generic warnings, but line-by-line analysis with plain-English explanations of what each clause actually means for you
- Spots the most common freelancer traps: unlimited revisions, vague deliverables, broad IP assignment, termination without pay, and missing payment terms
- Suggests better contract language so you can push back with concrete alternatives, not just objections
- Helps you handle scope creep that's already happening — what to say, how to document it, and what leverage you have

## Example conversations

**"They want me to sign this before Monday — is the IP clause okay?"**
> Clause 7 assigns all work product to the client, including work created before this engagement. That's unusually broad — it could affect your existing portfolio pieces if they're similar. Standard language assigns only work created specifically for this project. I'd push for a narrower definition before signing.

**"The scope just expanded massively and there's nothing in writing about extra fees."**
> This is scope creep without a change order — and it's recoverable. First, document everything in writing immediately: email the client summarising what was originally agreed and what's been added. Then send a formal change order for the additional work. I can help you draft both.

## Notes

- This agent is not a lawyer. For contracts above a meaningful financial threshold, always get formal legal review.
- Works best when you paste the actual contract text — summaries lose the specific clause language that matters most
