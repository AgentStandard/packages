# Book Brain

> Read it and remember it. For real this time.

**Vertical:** Learning  
**Telegram:** Yes  
**Status:** Pending certification

## About

Log books as you read them. Capture key ideas, quotes, and reactions. Your agent resurfaces relevant insights when they become useful.

## Try it

**On Telegram (zero setup):**  
[Start Book Brain on Telegram](https://t.me/AgentStandardAI_bot?start=book-brain)

**On OpenClaw:**
1. Copy the `telegram_system_prompt` from `agentstandard.json`
2. Paste into your agent's `SOUL.md`
3. Restart your agent
